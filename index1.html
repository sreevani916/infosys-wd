  • If I lost any company asset, whom should I reach out?

    In IT Asset Management, when an asset is reported to be lost or stolen, the first step is to file an official police report in the country where the incident took place.

    This is typically done by the custodian of the asset (end user), and should be done immediately after an incident has occurred.

  • I am not able to connect office network, should I raise a ticket for this?

    1. Make sure Wi Fi is turned on. Select the "No internet connection" icon on the right side of the taskbar, and make sure Wi-Fi is turned on.

    2. Also, make sure Airplane mode is turned off. If you still face the same problem please reach out to the IT team.
